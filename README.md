# 动态井字棋

一个永不分平的井字棋游戏，支持双人模式。

## 游戏规则

### 两阶段玩法

#### 第一阶段：放置阶段
- 玩家 X 先手，双方轮流在棋盘上放置自己的棋子
- 每位玩家最多放置 3 枚棋子
- 当 6 枚棋子全部放置完毕后，进入移动阶段

#### 第二阶段：移动阶段
- 玩家选择自己的一枚棋子
- 将该棋子移动到任意空格
- 先连成三连者获胜

### 胜负判定
- 横、竖、斜任意方向连成三子即获胜
- 由于移动阶段可以不断调整位置，理论上永远不会出现平局

## 操作说明

1. **放置棋子**：直接点击空白的格子即可放置当前回合的棋子
2. **移动棋子**：
   - 第一步：点击你要移动的棋子（会高亮显示）
   - 第二步：点击目标空格完成移动
   - 点击已选中的棋子可取消选择
3. **重新开始**：游戏结束后，点击"再来一局"按钮即可重置游戏

## 技术栈

- HTML5
- CSS3（包含动画效果）
- JavaScript（原生实现，无依赖）

## 运行方式

直接在浏览器中打开 `index.html` 文件即可开始游戏。
